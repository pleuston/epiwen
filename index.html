Epiwen · EpiDoc generator for Chinese stone inscriptions

A client-side, bilingual (EN / 中), dependency-free web tool that produces valid EpiDoc / TEI records for Buddhist stone-sutra inscriptions of North China — cliff-carved sutras, sutra steles, sutra pillars, and cave-wall inscriptions. Developed for the Altergraphy / Epiwen encoding project; modelled on the Hamburg Metadata Generator for EpiDoc.

Nothing is uploaded — the XML is built live in the browser. Records are contributed via GitHub pull requests and reviewed before they appear in this catalog.

Workflow

1

Adding a new inscription

  1. Open the Editor.
  2. Fill in metadata, physical description, date, and text(s). Use Add text for monuments with multiple faces.
  3. Click ① Copy XML — the full EpiDoc record is on your clipboard.
  4. Click ② Propose to GitHub — the web editor opens at records/your-file.xml.
  5. Paste the XML and click Propose new file. GitHub forks the repo and opens a pull request automatically.
2

Editing an existing inscription

  1. Find the monument in the Catalog below.
  2. Click Preview to inspect the current XML.
  3. Click Copy XML to copy it to your clipboard.
  4. Click Edit on GitHub — the web editor opens the existing file; paste your revised version.
  5. Submit a pull request. A project editor reviews and merges.
