Persons

Person Authority Sources

Person authority records in this corpus link out to the following external authorities via their cbdb, dila_authority, viaf, gnd and wikidata identifiers.

CBDB — China Biographical Database

中國歷代人物傳記資料庫（哈佛大學）
Harvard prosopography API

The standard prosopographical database of Chinese historical figures (c. 7th century BCE – 1949), with offices, kinship, social associations, and place-of-origin. Offers a public query interface and an API.

projects.iq.harvard.edu/cbdb

DILA Person Authority

法鼓山佛教文獻數位典藏 — 人名規範
Buddhist Chinese historical

Authority file of Buddhist monks, donors and historical persons in East Asia, maintained by the Dharma Drum Institute of Language and Culture — the primary reference for clerical names and dates in this corpus.

authority.dila.edu.tw/person/

Shanghai Library Open Data

上海圖書館開放數據平台
Shanghai linked data API

Shanghai Library's linked-data platform providing authority records for Chinese persons (as well as organisations and place names), cross-referenced with national standards and international authorities. Offers a REST search API and a SPARQL endpoint returning person records as RDF/JSON-LD. Authentication via a free API key (register at the developer portal).

data.library.sh.cn  ·  API docs

VIAF — Virtual International Authority File

虛擬國際規範檔
international library aggregator

Aggregates personal-name authority records from national libraries worldwide into clustered identifiers — the interoperability anchor linking a person to LC, BnF, NDL and other library authority files.

viaf.org

GND — Gemeinsame Normdatei

德國整合規範檔
Germany library scholarly

The Integrated Authority File of the German National Library, widely used in Western sinology and digital-humanities projects. Provides stable URIs for persons cited in the scholarly literature.

explore.gnd.network

Wikidata

維基數據
open linked data QID

Wikidata QIDs serve as a lightweight interoperability anchor for person records, linking to Wikipedia, VIAF, CBDB and other systems via the wikidata identifier on person authority records.

wikidata.org
